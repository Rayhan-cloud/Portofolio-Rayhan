CSS (Cascading Style Sheets)

Pengenalan CSS

CSS digunakan untuk mengatur tampilan dan gaya pada halaman web. Saya belajar CSS agar dapat mengubah warna, layout, font, dan animasi dari elemen HTML secara efisien.

Progres Belajar

  • ✔️ Styling dasar: warna, margin, padding, font
  • ✔️ Flexbox dan Grid layout
  • ✔️ Pseudo-class dan media query
  • ✔️ Animasi dasar dan efek transisi

Tools yang Digunakan

Saya menggunakan Visual Studio Code dan Live Server untuk melihat perubahan desain secara langsung di browser.

Tujuan Belajar

Memperindah halaman web dan membuat tampilan yang responsif di berbagai perangkat agar pengunjung merasa nyaman saat mengakses situs saya.

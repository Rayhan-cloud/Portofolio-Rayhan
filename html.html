HTML (HyperText Markup Language)

Pengenalan HTML

HTML adalah fondasi dari setiap halaman web. Saya mulai belajar HTML sebagai langkah awal menjadi web developer, karena HTML sangat penting dalam struktur konten di internet.

Progres Belajar

  • ✔️ Struktur dasar dokumen HTML
  • ✔️ Elemen penting: heading, paragraph, list, table
  • ✔️ Form input dan validasi sederhana
  • ✔️ Menyematkan gambar, video, dan link

Tools yang Digunakan

Saya menggunakan Visual Studio Code untuk membuat dan menulis struktur HTML, serta browser Chrome untuk mengecek hasilnya secara langsung.

Tujuan Belajar

Menjadi dasar kuat sebelum mendalami CSS dan JavaScript agar mampu membangun website yang baik dari sisi struktur dan konten.

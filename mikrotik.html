Konfigurasi Mikrotik

Pengenalan Mikrotik

Mikrotik adalah sistem operasi dan perangkat lunak yang digunakan untuk menjadikan komputer sebagai router jaringan yang handal. Saya belajar menggunakan Winbox untuk melakukan konfigurasi perangkat Mikrotik.

Pengalaman dan Konfigurasi

  • ✔️ Konfigurasi dasar IP Address dan DHCP Server
  • ✔️ Konfigurasi NAT untuk internet sharing
  • ✔️ Manajemen bandwidth menggunakan Queue
  • ✔️ Setting firewall dan keamanan dasar

Tools dan Perangkat

Saya menggunakan Winbox, MikroTik RouterOS, dan perangkat seperti RB750Gr3 dalam praktik langsung.

Tujuan Belajar

Meningkatkan pemahaman tentang manajemen jaringan dan keamanan dasar dalam lingkungan kecil seperti sekolah atau kantor kecil.

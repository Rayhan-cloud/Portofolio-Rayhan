Konfigurasi Ruijie

Pengenalan Ruijie

Ruijie adalah brand perangkat jaringan yang banyak digunakan di lingkungan pendidikan dan perusahaan karena kemudahan manajemen serta fiturnya yang lengkap. Saya telah mempelajari konfigurasi menggunakan Ruijie Cloud dan aplikasi Ruijie Reyee.

Pengalaman Konfigurasi

  • ✔️ Konfigurasi dasar access point (SSID, password)
  • ✔️ Penggunaan Ruijie Cloud untuk monitoring jaringan
  • ✔️ Konfigurasi VLAN dan pembagian trafik
  • ✔️ Setup login portal hotspot

Tools dan Perangkat

Saya menggunakan aplikasi Ruijie Cloud, Ruijie Reyee App, serta perangkat access point RG-RAP2200 dan switch layer 2 dari Ruijie.

Tujuan Belajar

Untuk memahami sistem manajemen jaringan berbasis cloud dan membangun jaringan yang efisien dan mudah dipantau dari mana saja.
